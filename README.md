# Society Maintenance Tracker

A web application for managing maintenance complaints, notices, and
administrative operations within a residential society.

## Features

-   Resident authentication
-   Raise and track complaints
-   Upload complaint photos
-   Admin complaint management
-   Complaint status updates and history tracking
-   Society notices and announcements
-   Role-based access control
-   Supabase authentication and database integration

## Tech Stack

-   React
-   TypeScript
-   Supabase
-   Tailwind CSS
-   Vite
